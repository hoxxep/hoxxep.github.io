Snarl

Growl-style notifications for your web app.

v0.1.1

Snarl Notifications

Snarl intends to make it quick and easy to include gorgeous and functional Mac/Growl-style notifications in your web app. It supports desktop, touch/tablet, and mobile devices; has raw JS and LESS/CSS source files; and supports custom timeouts and notification click callbacks.

Basic Demo

Adding a notification is really simple with snarl. Specify a title and some text (as a description) and your notification will appear!

{% highlight javascript %} $('#basic-demo').click(function() { Snarl.addNotification({ title: 'Snarl', text: 'Hello, world!' }); }); {% endhighlight %}

Timeouts

Snarl supports custom timeouts specified in milliseconds or even having no timeouts at all.

Custom Timeout

{% highlight javascript %} $('#custom-timeout-demo').click(function() { Snarl.addNotification({ title: 'Custom Timeouts', text: 'This notification has an 8000ms timeout!', timeout: 8000 }); }); {% endhighlight %}

No Timeout

Notifications with no timeouts must either be manually dismissed by the user, or can be removed by your code once an event is complete. Updating notifications is done using the ID string returned by Snarl.addNotification().

{% highlight javascript %} var noTimeoutNotification = null; $('#no-timeout-open').click(function() { if (Snarl.isDismissed(noTimeoutNotification)) { noTimeoutNotification = Snarl.addNotification({ title: 'No Timeout', text: 'Close the notification by pressing either the dismiss button on the page or in the notification.', timeout: null }); } }); $('#no-timeout-close').click(function() { Snarl.removeNotification(noTimeoutNotification); }); {% endhighlight %}

Update Notifications

Because notifications are track/modifiable with their IDs, notifications can be updated, reopened, and removed. All options on a notification can be updated later by resupplying the options you wish to change with the Snarl.editNotification(id, options) function.

{% highlight javascript %} /** * Take user input */ function options() { var title = $('#update-notification-title').val(), text = $('#update-notification-text').val(), timeout = $('#update-notification-timeout').val(), dismissable = $('#update-notification-dismissable').prop('checked'); if (title === '') title = 'Notification Title'; if (text === '') text = 'Notification Text'; if (timeout === '') timeout = null; return { title: title, text: text, timeout: timeout, dismissable: dismissable }; } /** * Snarl-related code */ var notification = null; $('#update-notification-open').click(function() { if (!Snarl.exists(notification)) { notification = Snarl.addNotification(options()); } else { Snarl.editNotification(notification, options()); } }); $('#update-notification-close').click(function() { Snarl.removeNotification(notification); }); {% endhighlight %}

Click Actions/Callbacks

If the user clicks a notification, an action can be run. Either a callback function can be executed; a url can be visited; or nothing. These are specified via the action option. If a function is provided it will be used as a callback, if a string is provided then that url will be visited, or if null is used then no action will occur. The default value is action: null.

Action: String/Url

Visit a url when the user clicks the notification. Note: currently does this by setting window.location.

{% highlight javascript %} $('#snarl-callback-string').click(function() { Snarl.addNotification({ title: 'String Callback', text: 'Click this notification to visit a url', action: 'http://github.com/hoxxep/Snarl' }); }); {% endhighlight %}

Action: Callback Function

This example makes the notification update itself once clicked. The function is passed the notification id as an argument, and has the potential to work with JS MVC frameworks too. Note that if the dismiss button is pressed, then no callback is triggered (subject to change).

{% highlight javascript %} $('#snarl-callback-function').click(function() { var id = Snarl.addNotification({ title: 'Snarl Notification #' + (Snarl.count + 1), text: 'Hello, world! Click this notification.', timeout: null, action: function(id) { Snarl.editNotification(id, { text: 'You clicked the notification! Click once more, click the dismiss button, or wait 8000ms to close it.', timeout: 8000, action: function(id) { Snarl.removeNotification(id); } }); } }); }); {% endhighlight %}

Messages Demo

The messages demo is a simple example using jQuery to demo how Snarl notifications could look in a more realistic usage example.

{% highlight javascript %} var typingNotification = null; $('#name').focusout(function() { typingNotification = null; if ($('#name').val()) { Snarl.addNotification({ title: $(this).val(), text: 'Signed in' }); } }); $('#message').keypress(function() { if ($('#name').val()) { if (typingNotification === null) { typingNotification = Snarl.addNotification({ title: $('#name').val(), text: 'Is typing...', timeout: 2000 }); } else { Snarl.reOpenNotification(typingNotification); } } }); $('#send-message').click(function() { if ($('#name').val() && $('#message').val()) { Snarl.removeNotification(typingNotification); typingNotification = null; Snarl.addNotification({ title: $('#name').val(), text: $('#message').val(), timeout: 5000 }); $('#message').val(''); } else { Snarl.addNotification({ title: 'Error', text: 'No name or message entered!' }); } }); {% endhighlight %}

Snarl Notification Reference

Common Function Arguments

Argument: id

The id is the string which maps to a Snarl notification. ids are generated and returned automatically when using .addNotification(options).

Argument: options

title: String - title of the notification.

text: String - description part of the notification.

timeout: Integer/null

  • Integer: timeout of notification in milliseconds.
  • null: no timeout.

action: String/Function/null

  • String: url of page to visit if notification is clicked.
  • Function: callback function to run on notification click. Is passed the notification id as a parameter.
  • null: no callback/action when clicked.

dismissable: true/false whether the notification is dismissable

Default Options

{% highlight javascript %} Snarl.defaultOptions => { title: '', text: '', timeout: 3000, action: null, dismissable: true } {% endhighlight %}

Example Options

Note: for user friendliness, most notifications should be dismissable (and they are by default), hence why dismissable is omitted in this example.

{% highlight javascript %} var options = { title: 'Snarl Notification', text: 'Hello, world! This is an example notification.', timeout: 5000, action: function(id) { // do relevant action in web app console.log('Snarl notification ' + id + ' was clicked!'); } }; {% endhighlight %}

Snarl Methods

.setDefaultOptions(options)

Set's the default options to be used when adding a new notification.

{% highlight javascript %} // Example: Set default timeout to null Snarl.setDefaultOptions({ timeout: null }); {% endhighlight %}

.addNotification(options) : String id

Open a new notification with the specified options. Returns a string that is the notification id.

{% highlight javascript %} // Example: Add new notification var notification = Snarl.addNotification({ title: 'Snarl Notification', text: 'Example notification text', action: 'http://hoxxep.github.io/Snarl' }); {% endhighlight %}

.editNotification(id, options)

Edit an open notification. Only supply the options that you want to update, the rest will stay the same. If the notification has been dismissed it will re-open the notification with the previous timeout setting.

{% highlight javascript %} // Example: Edit the description on our new notification Snarl.editNotification(notification, { text: 'I\'ve edited the notification text!' }); {% endhighlight %}

.reOpenNotification(id)

Attempt to re-open a notification otherwise, if the notification is still open, reset the timeout.

{% highlight javascript %} // Example: Re-open the notification if it was dismissed Snarl.reOpenNotification(notification); {% endhighlight %}

.removeNotification(id) : Boolean successful

Dismiss a notification. Returns true if the notification was open and false if it was already dismissed or did not exist. Note removed notifications can still be re-opened or edited, this merely dismisses a notification. Will probably be refactored to .dismissNotification in the near future.

{% highlight javascript %} // Example: Manually dismiss the notification Snarl.removeNotification(notification); {% endhighlight %}

.isDismissed(id) : Boolean dismissed

Checks if a notification is open. Returns true if the notification has been dismissed or does not exist.

{% highlight javascript %} // Example: Because we dismissed/removed the notification .isDismissed is true Snarl.isDismissed(notification) // => true {% endhighlight %}

.exists(id) : Boolean exists

Check if a notification id has been used. Returns true if notification has been opened with that id even if it's been dismissd.

{% highlight javascript %} // Example: Notification with our id still exists, even after being // dismissed/removed. This means the notification can still be re-opened. Snarl.exists(notification) // => true {% endhighlight %}

.setTitle(id, title)

Set the title of a notification. Shorthand for .editNotification(id, {title: title}).

{% highlight javascript %} // Example: Change the title of our notification // Note: this will re-open our notification Snarl.setTitle(notification, 'Changed the title!'); {% endhighlight %}

.setText(id, text)

Set the text of a notification. Shorthand for .editNotification(id, {text: text}).

{% highlight javascript %} // Example: Change the text of our notification // Note: this will re-open and/or reset the timeout on our notification // to it's current setting. Snarl.setTitle(notification, 'Changed the text!'); {% endhighlight %}

.setTimeout(id, timeout)

Set the timeout of a notification. Shorthand for .editNotification(id, {timeout: timeout}).

{% highlight javascript %} // Example: Remove the timeout altogether Snarl.setTitle(notification, null); {% endhighlight %} {% highlight javascript %} // Example: Make timeout 8000ms Snarl.setTitle(notification, 8000); {% endhighlight %}
